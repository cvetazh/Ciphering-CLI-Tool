# Ciphering-CLI-Tool
1. To run script 
$ node my_ciphering_cli -c "C1-C1-R0-A" -i "./input.txt" -o "./output.txt"
2. To set encoding use [require]
-c "..." or --config "..."
3. To set the input file [optional] use 
-i "..." or --input "..."
4. To set the output [optional] file use
-o "..." or --output "..."
